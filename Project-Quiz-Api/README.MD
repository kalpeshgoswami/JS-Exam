# 🧠 Quiz App (JavaScript)

A simple and interactive Quiz Application built using **HTML, CSS, Bootstrap, and JavaScript**.  
This app presents multiple-choice questions, tracks your score, and shows a detailed result summary at the end.

---

## 🚀 Live Demo
https://heroic-gumption-587963.netlify.app/

---

## 📌 Features

- ✅ Multiple-choice questions
- ⏱️ 30-second timer for each question
- 👉 Auto-skip when time runs out
- 📊 Score tracking
- 📋 Detailed answer review after quiz
- 🎨 Clean UI with Bootstrap styling

---

## 🛠️ Technologies Used

- CSS3

- JavaScript (Vanilla JS)

## ⚙️ How It Works

1. Quiz loads automatically on page start
2. Each question has a **30-second timer**
3. User selects an answer:
   - Moves to next question instantly
4. If time runs out:
   - Question is skipped
5. After all questions:
   - Final score is displayed
   - Full answer review is shown
